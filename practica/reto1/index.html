Etiquetas em, strong, code, q y blockquote

La etiqueta em se utiliza para dar énfasis a una palabra. Tiene un uso similar a la etiqueta strong que usaremos cuando queramos marcar una palabra como importante: Por ejemplo

El perro es un animal muy fiel considerado en muchos casos el mejor amigo del hombre

Podemos encontrar la siguiente referencia a la etiqueta strong en www.campusmvp.es :

La etiqueta strong realmente no está diseñada para dar negrita, aunque el resultado sea en realidad ese. Esta etiqueta es semántica, y por lo tanto sirve para dar contexto al contenido de la página. En este caso la eetiqueta sirve para indicar que lo que va dentro de ella reviste importancia o urgencia

Para marcar palabras o fragmentos de texto que hacen referencia a código de programación usaremos la etiqueta code

La etiqueta q la utilizaremos para marcar citas cortas. Cuando queremos hacer referencia a un bloque de texto extroido de otra página

Etiquetas marksmall, sub, sup y address

Podemos resaltar un fragmento de texto usando mark. Por ejemplo, este texto está resaltado

Si queremos marcar un fragmento de texto para que salga mas pequeño que el resto del parrafo usando la etiqueta small. Por ejemplo vemos que este texto es mas pequeño que el resto

También podemos marcar texto como sub-indice o super-indice con las etiquetas suby sup. Por ejemplo: palabra subindice o palabra superíndice

Finalmente, si queremos marcar un texto para indicar al navegador que es dirección podemos usar la etiquetaaddress. Por ejemplo: Vivo en

calle calvario número 12
